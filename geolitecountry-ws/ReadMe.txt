GeoLite Country ASP.NET Web Service
-----------------------------------

Build
-----

- .NET SDK 1.1 is required
- Run build.bat in source folder to build

Deploy
------

- Deploy GeoIP.dat and GeoLiteCountry.asmx together to any folder on web server
- Deploy GeoLiteCountry.dll to bin folder

API
---

- Just load GeoLiteCountry.asmx directly from browser to see the API

Access Key
----------

- The access key is hard coded in GeoLiteCountry.asmx file
- Just remember to change the access key to your own
